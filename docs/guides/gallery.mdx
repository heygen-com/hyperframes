Browse all built-in templates to find the right starting point for your video. Each preview shows the template rendered at its native resolution.

To scaffold a project from any template:

```bash Terminal
npx hyperframes init my-video --template <name>
```

<Tip>
  Looking for a minimal starting point? Use `blank` — it gives you an empty composition with just the scaffolding, no visual design. Run `npx hyperframes init my-video --template blank` to get started from scratch.
</Tip>

## Choosing a Template

| Template | Style | Best for |
|----------|-------|----------|
| `warm-grain` | Organic, textured | Lifestyle, branding, editorial |
| `play-mode` | Energetic, elastic | Social media, product launches |
| `swiss-grid` | Clean, structured | Corporate, data, technical |
| `vignelli` | Bold, typographic | Headlines, announcements |
| `kinetic-type` | Dramatic type | Promos, intros, title cards |
| `decision-tree` | Diagrammatic | Explainers, tutorials |
| `product-promo` | Multi-scene | Product showcases, demos |
| `nyt-graph` | Editorial data | Data stories, reports |
| `blank` | Minimal scaffolding | Full control, agent-generated |
